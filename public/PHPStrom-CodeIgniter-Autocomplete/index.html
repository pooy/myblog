发表于 2016-12-10 | 分类于 Testing | | 阅读次数 |

安装步骤

克隆 phpStorm-CC-Helpers 项目

打开 phpStorm 右键单击 External Libraries > Configure PHP Include Paths > 添加 phpStorm-CC-Helpers 的路径，如下图所示：

标记 PHPStrom 中的 CodeIgniter 框架下的三个文件 「Mark as Plain Text」

  • /system/core/Controller.php
  • /system/core/Model.php
  • /system/database/DB_active_rec.php

打开你的 CI 项目，在你项目里面测试，如下图，是否发现能看到代码提示了？
