背景

之前是打算使用 github 的 pages 服务，国内访问还是太慢了，索性就用国内 coding 厂商提供的 pages 服务吧。在使用这个服务之前，一直使用着 coding 提供的 git 私有 服务，最近 2 年在上面管理着十来个客户的项目，一路走过来用的比较放心。
